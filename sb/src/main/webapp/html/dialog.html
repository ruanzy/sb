$.dialog({
	title: 'title',
	width: 300,
	height: 200,
	content: 'html',
	buttons: [
		{
			text: '同意',
			click: function () {
				this.title('你同意了');
			}
		},
		{
			text: '不同意',
			click: function () {
				alert('你不同意');
			}
		}
	]
});

配置说明

属性 类型 描述
title string 标题
width int 宽度
height int 高度
content string html内容
buttons array

自定义按钮, 配置项:

text: 按钮文本

click: 按钮点击事件

buttons:[
	{
		text: '同意',
		click: function(){
			alert('同意');
		}

		click: function(){
			alert('不同意');
		}
	}
