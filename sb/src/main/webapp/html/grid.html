使用示例

<div id="datalist"></div>
$('#datalist').grid({
	url : 'user/list',
	columns: [
		{ header: '用户名', field: 'username'},
		{ header: '注册时间', field: 'regtime', align: 'center', width: 200},
		{ header: '操作', field: 'op', render : opRender, align: 'center', width: 200}
	]
});
function stateRender(v, r, rowid){
	return (v == 1)? 'a': 'b';
}

配置说明

属性 类型 描述
columns array grid的列column的配置对象.更多细节请参见列column属性。
url string 从远程站点请求数据的 URL
data array 要加载的数据
sm string 实现单选、多选 ('s':单选,'m':多选)
pagesize int 分页的每页记录数(-1表示不分页)

列属性说明

属性 类型 描述
header string 列的标题文本
field string 列的字段名
width int 如果未定义，则宽度会自动扩展以适应它的内容
align string 指示如何对齐该列的数据，可以用 'left'、'right'、'center'
render function

单元格的格式化函数，需要三个参数：

value：字段的值

rowData：行的记录数据

rowid：行的索引

function stateRender(v, r, rowid){
	return (v == 1)? 'a': 'b';
}

方法

名称 参数 描述
getSelected none 返回所有选中的行，当没有选中的记录时，将返回空数组
reload param 重新加载数据
setQueryParam param 设置查询参数
